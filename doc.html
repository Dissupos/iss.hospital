Implementation

Project consists of two main parts: frontend and backend. Frontend has been implemented using popular libraries, such as: AngularJS, Bootstrap, Gulp, NPM, Bower. AngularJS is used to create the main structure of the project. It is SPA(Single Page Application), so every route of the application is actually the main page. Backend has been implemented using Java, Spring Boot, Spring Security, Spring JPA. Spring JPA is used for the implementation of the data access layer of an application. Spring Security is used for checking user rights. Communication between frontend and backend is made in the form of RESTful Web Service. Every time when user changes anything on the front end, a request is sent to the backend to change it in the database. For example: When doctor is creating new patient, a request "POST /api/staff/" is sent and backend handles it.

Installation

For installation you will need: * Install: npm, bower, gulp * Install: java sdk, maven * Create empty MySQL database * Go to the /client directory and run npm install * Go back to the root directory of the project * In the root directory of the project run command: mvn package * Go to the /target directory * In the /target directory run command: java -jar hospital-0.0.1-SNAPSHOT.jar --spring.datasource.url=[JDBC_URL] --spring.datasource.username=[USERNAME] --spring.datasource.password=[PASSWORD] where JDBC_URL is url of the database, USERNAME and PASSWORD are auth data for database.
